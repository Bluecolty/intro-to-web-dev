This weeks project requirements

This weeks assignment/project requirements needed 3 different ways to style HTML tags, including


  • The element selector, which styles the HTML tag its linked to.
  • The ID selector, which creates a CSS styling ID that can be added to different HTML elements/tags.
  • The class selector, which links a CSS class to an HTML tag.
