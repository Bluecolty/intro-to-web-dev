Project and Course Information

This website is the polished culmination of much of the content learned in CSCI 121 - Intro to Web Development.

Course Description for CSCI 121

Designed to introduce web programming to students who are not computer science majors. Topics include web-page design, scripting languages, graphics animation, image handling, event handling, document object model and graphical interface control objects. No credit toward computer science major. Offered periodically.

Project Information

For my final project, I decided to create a portfolio showcasing my work both in this class, and in my AENG 252 - Web Publishing Systems class. This website incorporates HTML, CSS, and JavaScript to cover all of the content areas we covered in class.

For my project, I settled on a portfolio that would detail 3 areas of my life. My website details my portfolio of projects worked on in this class, my woodworking projects done in AENG 130 and AENG 271, and it details my phone repair adventures. The phone repair was designed to be more content filler than anything.
